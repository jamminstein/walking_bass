# walking_bass

> Infinite upright walking bass for drum practice

## Controls

- **E1** — change screen page (play/tone/trainer)
- **E2** — adjust tempo
- **E3** — select progression
- **K2** — start / stop
- **K3** — reset and play

## Grid

16×8 monome grid for chord progression editing. Rows 1-6 select chord roots, row 7 toggles quality, row 8 has transport controls.

## Requirements

- norns
- grid (16×8 monome, optional but recommended)
- MIDI device (optional, for external sync/output)
- engine: UprightBass

## Install

```
;install https://github.com/jamminstein/walking_bass
```
