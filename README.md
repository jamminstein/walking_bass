# walking_bass

An infinite upright walking bass for drum practice. Features a custom physical-modeled UprightBass engine, grid-based chord progression builder, and tempo trainer with multiple chorus forms.

## Install

`;install https://github.com/jamminstein/walking_bass`

## Controls

**Norns:**
- K2: Start / Stop playback
- K3: Reset and play
- E1: Page navigation
- E2: Tempo (or grid root note)
- E3: Progression (or grid quality/scale)

**Grid (16x8):**
- Rows 1-6: Chord slots for progression building
- Row 7: Quality toggle (major/minor and extensions)
- Row 8: Transport controls and navigation

## Features

- Custom physical-modeled upright bass engine
- Infinite walking patterns with tempo trainer
- Grid-based chord progression builder
- Multiple chorus forms (verse, bridge, etc.)
- Count-in for metronome practice
- MIDI output support
- Bandleader edition with advanced count-in and tempo training

## Requirements

- norns
- 16x8 grid
- Engine_UprightBass engine (included)
